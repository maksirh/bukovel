# Hotel Images — Готель Затишний двір
Фотограф: Михайло Осачук

## Структура папок

| Папка | Призначення |
|-------|-------------|
| `exterior/` | Зовнішній вигляд, фасад, територія |
| `rooms/` | Номери, спальні, ванні кімнати |
| `interior/` | Інтер'єр, лобі, спільні приміщення |

## Як розсортувати

Всі фото знаходяться у `_all/` у порядку галереї (001.png, 002.png...).
Перегляньте і перемістіть у відповідні тематичні папки.

**Всього фото:** 100
**Якість:** large (оригінал, ~4000×2668 px)
